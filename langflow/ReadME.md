# Langflow

## Install Langflow

Use `uv`, create a venv with `uv venv .venv --python 3.11` and follow these [instructions](https://docs.langflow.org/get-started-installation#install-and-run-the-langflow-oss-python-package)
Possibly install [Microsoft Visual Studio](https://visualstudio.microsoft.com/de/downloads/) and especially support for `Microsoft Visual C++`. 
Use `uv run langflow run` to start Langflow.

Nutze `uv`, erstelle eine venv mit `uv venv .venv --python 3.11` und befolge diese [Anweisungen](https://docs.langflow.org/get-started-installation#install-and-run-the-langflow-oss-python-package), um Langflow via `uv` zu installieren.
Möglicherweise ist eine Installation von [Microsoft Visual Studio](https://visualstudio.microsoft.com/de/downloads/) und insbesondere die Unterstützung für „Microsoft Visual C++“ nötig. 
Verwende `uv run langflow run`, um Langflow zu starten.

## Langflow Flows importieren

### Import von JSON-Flows

1. Öffne Langflow in deinem Browser
2. Klicke auf **"New Project"** oder das **"+"** Symbol
3. Wähle **"Upload"** oder **"Import Flow"**
4. Wähle die gewünschten `.json` Dateien aus diesem Repository
5. Die Flows werden automatisch geladen und sind bereit zur Verwendung

### Verfügbare Flows

Die JSON-Dateien in diesem Repository enthalten vorkonfigurierte Langflow-Workflows, die direkt importiert und verwendet werden können.

### MCP-Konfiguration

**Wichtiger Hinweis:** Die MCP (Model Context Protocol) Konfigurationen müssen separat eingerichtet werden. Detaillierte Anleitungen zur MCP-Konfiguration findest du im Obsidian Vault dieses Repositories.
