# Langflow Flows importieren

## Import von JSON-Flows

1. Öffne Langflow in deinem Browser
2. Klicke auf **"New Project"** oder das **"+"** Symbol
3. Wähle **"Upload"** oder **"Import Flow"**
4. Wähle die gewünschten `.json` Dateien aus diesem Repository
5. Die Flows werden automatisch geladen und sind bereit zur Verwendung

## Verfügbare Flows

Die JSON-Dateien in diesem Repository enthalten vorkonfigurierte Langflow-Workflows, die direkt importiert und verwendet werden können.

## MCP-Konfiguration

**Wichtiger Hinweis:** Die MCP (Model Context Protocol) Konfigurationen müssen separat eingerichtet werden. Detaillierte Anleitungen zur MCP-Konfiguration findest du im Obsidian Vault dieses Repositories.
