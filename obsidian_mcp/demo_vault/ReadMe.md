# Obsidian MCP und Workshop Inhalte

## Verwendung

**Wichtiger Hinweis:** Die Tutorials und Inhalte in diesem Repository sind für die Anzeige in Obsidian optimiert. Für die beste Erfahrung wird empfohlen, die Markdown-Dateien direkt in Obsidian zu öffnen, anstatt sie in einem Code-Editor wie VS Code zu betrachten.

### Installation und Setup

1. Installiere [Obsidian](https://obsidian.md/) falls noch nicht vorhanden
2. Öffne Obsidian und wähle den `demo_vault` als Vault aus
3. Die Workshop-Inhalte werden nun mit allen interaktiven Features und der korrekten Formatierung angezeigt

## Warum Obsidian?

Die Tutorials nutzen spezifische Obsidian-Features wie:
- Interne Verlinkungen zwischen Notizen
- Eingebettete Inhalte und Graphenansicht
